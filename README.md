# Lighter

Lighter is a high-performance Minecraft: Pocket Edition server core based on Genisys 0.13.x.

## What is this?

This is an MCPE server core that offers a great amount of amazing functions!

## Features

- **PHP 8.0 Supports**  
- **Better AI** - Wolf, Ghast, Squid, Blaze, Villager, Iron Golem, Rabbit...
- **Completed Enchantment System** - EnchantingTable (FishingRod, Bow, Counting BookShelf), Anvil (Rename, Enchanting, Fixing), All Enchantment Properties Available
- **Fishing System** - Real Treasure Table according to the enchantment properties of fishing rod
- **Better Generator** - Forest, Desert Temple, Tropical Rainforest...
- **Multi-Version Support** - (From 0.12.x to 0.13.x), Protocol 34 ~ 39
